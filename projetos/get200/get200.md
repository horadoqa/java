# GET 200

Executando uma requisição e retornando seu Status Code

## Compilando

```bash
javac get200/HttpStatusChecker.java
```

## Executando

```bash
java get200/HttpStatusChecker https://www.google.com
```

## Resposta

```bash
Status code: 200
```

```bash
java get200/HttpStatusChecker http://www.globo.com 
```
